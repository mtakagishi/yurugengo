フレーム問題
==========================================
.. glossary::
    フレーム問題 : ふ

人工知能に何かを教え込むときにどこまで教えるべきか論理的に決めきれない問題

.. tip:: 
  「ガヴァガイ問題」と似ている。赤ちゃんの生まれながらに持つバイアスがヒントになるのかも。
