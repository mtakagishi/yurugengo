エスペラント
====================
.. glossary::
  エスペラント : え
  エスペランティスト : え

ルドヴィコ・ザメンホフとその弟子（協力者）が考案・整備した人工言語(wikipedia)。エスペラント語を話す人のことをエスぺランティストと呼ぶ
