specifier
==========================================
.. glossary::
    specifier : A-Z
    スペシファイヤー : す

Xバー理論（ふたまたにょきにょき）に登場する指定部のこと。通常会話で当たり前に使うと言語学に興味のない人間を置き去りにする効果を発揮する。言語学に対し特段の興味がない :doc:`ゆる言語学ラジオ` の聞き手パーソナリティのトラウマとなっている。
