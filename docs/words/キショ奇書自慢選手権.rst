キショ奇書自慢選手権
==========================================
.. glossary::
    キショ奇書自慢選手権 : き

蔵書のセンスを誇りに思う読書好きが一つや二つは持っているキショ奇書を惜しみなく自慢し合う企画。ビブリオバトルっぽくなりがち。

.. tip:: 
  * キショ奇書とは、キショいと奇書をかけたダジャレ。
  * 蔵書にある変わった本、ユニークな本、すごい本。
  * 衝動でカートに入れがち、買いがち、読まないがち
