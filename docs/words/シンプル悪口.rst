シンプル悪口
==========================================
.. glossary::
    シンプル悪口 : し

単なる悪口。堀元氏の `教養（インテリ）悪口 <https://amzn.to/3Ko2vsc>`_ とは異なり捻りのなく直接的に悪く言う言葉。クソバカ、アホ、ゴミ、など。
