標高マウント・マウントマウント
==========================================
.. glossary::
    標高マウント : ひ
    マウントマウント : ま

住んでいる自宅の標高でマウントを取ること。

.. tip:: 
  長野県を案内する飯田さん、標高の話題の旅に「うちは1200mのところに家があるけどね」を連呼していたらしい。
