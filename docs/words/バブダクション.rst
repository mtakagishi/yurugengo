バブダクション
==========================================
.. glossary::
    バブダクション : は
    babduction : A-Z

baby-production または baby-abductionの略。赤ちゃんの言い間違いは「間違い」ではなく子供たちが生み出した推論の結果であり尊い。英語ではchild specific phonology(子供特有の音韻変化)と呼ぶこれらの現象に日本語として命名したい川原先生が生み出した造語。
