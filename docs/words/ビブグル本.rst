ビブグル本
==========================================
.. glossary::
    ビブグル本 : ひ

ミシュランのビブグルマンと本の合成語。価格以上の満足感が得られる料理としてのビブグルマンの古本屋で低価格帯で売られているけどメッチャおもろい本。
