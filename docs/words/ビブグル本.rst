ビブグル本
==========================================
.. glossary::
    ビブグル本 : ひ

ミシュランのビブグルマンと本の合成語。古本屋で100円で売られてるけどメッチャおもろい本。

.. tip::
  `ビブグルマン <https://guide.michelin.com/jp/ja/selection/japan/restaurants/bib-gourmand>`_ は「価格以上の満足感が得られる料理」を紹介するミシュランガイド
