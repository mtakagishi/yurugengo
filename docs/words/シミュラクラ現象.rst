シミュラクラ現象
==========================================
.. glossary::
    シミュラクラ現象 : し

3つの点ががあると人の顔に見える現象。「似ている物」を意味するラテン語"simulacrum"に由来。

.. tip:: 
  シミュラクラさんがみつけたというエポニム、、、だと堀元氏の勘違いがあった。
