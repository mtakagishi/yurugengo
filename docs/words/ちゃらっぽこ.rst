へっぽこ・ちゃらっぽこ
==========================================
.. glossary::
    ちゃらっぽこ : ち
    へっぽこ : へ

へっぽこの「ぽこ」はちゃらっぽこの「ぽこ」。ぽこはマイナスのイメージの状態を示す造語成分。

.. tip:: 
  ちゃらっぽこ とは、デタラメを言う人、嘘つき。面白い言葉なのに何回言っても覚えられない。
