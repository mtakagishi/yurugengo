コネクティングザドッツみ
==========================================================
.. glossary::
  コネクティングザドッツみ : こ
  コネドみ : こ
  コネドる : こ

* 知ってる知識（点）が結びついて線になる感覚・実感
* 略して「コネドみ」、またその感覚が発生したときは「コネドる」

.. tip:: 
  * コネクティングザドッツは、2005年スタンフォード大学でスティーブジョブズの演説による。
