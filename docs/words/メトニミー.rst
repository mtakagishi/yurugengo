メトニミー
==========================================
.. glossary::
    メトニミー : め

「示したいもの」の代わりに、近いイメージのものに結び付けて伝える比喩表現、

.. rubric:: メトニミーの例文
:赤ずきん: 少女のイメージ
:やかんが沸騰した: 沸騰するのはお湯
:鍋を食べる: 鍋でなく鍋の中身を食べる
:新聞を読んだ: 読むのは活字
:ここのレストラン美味しい: おいしいのは料理
