メトニミー
==========================================
.. glossary::
    メトニミー : め

* 近いイメージのものに結び付ける、置き換える比喩

.. rubric:: メトニミーの例文
:赤ずきん: 少女のイメージ
:やかんが沸騰した: 沸騰するのはお湯
:鍋を食べる: 鍋でなく鍋の中身を食べる
:新聞を読んだ: 読むのは活字
:ここのレストラン美味しい: おいしいのは料理
