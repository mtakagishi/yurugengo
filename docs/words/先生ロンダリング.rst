先生ロンダリング
==========================================
.. glossary::
    先生ロンダリング : せ

専門家にゲストに来てもらうことで、素人がアカデミックの二次創作を思う存分できる仕組み。ふざけちらかした話をしても専門家がきちんと回収。本当に間違っていることはその場で訂正してもらうことで、ふざけた印象そのものを拭ってもらえる。
