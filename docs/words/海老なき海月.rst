海老なき海月
==========================================
.. glossary::
    海老なき海月 : え

頼れるものがなくなって途方に暮れることの例え

.. tip:: 
  別名「借眼公」とも呼ばれるクラゲは、エビの目を借りて泳ぐという俗説があることから。
