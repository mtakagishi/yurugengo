周知の事実
==========================================
.. glossary::
    周知の事実 : し

初心者を置き去りにする一撃。専門的な話題の際に必要な前提知識を「周知の事実」とすることで面倒な説明を省略できる。

.. tip:: 
  一緒に酒を飲んでいるのに置いていかれると寂しい
