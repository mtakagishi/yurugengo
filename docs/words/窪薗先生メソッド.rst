窪薗先生メソッド
==========================================================
.. glossary::
  窪薗先生メソッド : く

* 無知は知識量を測るバロメータであるという考え方
* 理解できた領域があるから、理解できてない領域を再確認できる。

.. tip:: 
  * ヒトの知識量は無知の量で決まるという窪薗晴夫先生の著書に書かれた考え方。
  * 勉強すればするほど分からなくなる現象を受け入れ、無知である領域を再確認する。
