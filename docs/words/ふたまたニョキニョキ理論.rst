ふたまたニョキニョキ
==========================================
.. glossary::
    ふたまたニョキニョキ : ふ

1. 生成文法におけるXバー理論のこと。あらゆる文章を表現できる理論のため、明日から会話のたびに「はい、お前の発言、Xバー」「ナイス、Xバー」という返しができるようになる。いじめに使ってはいけない。

2. データ構造における二分探索木のこと。
