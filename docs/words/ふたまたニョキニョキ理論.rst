ふたまたニョキニョキ理論
==========================================
.. glossary::
    ふたまたニョキニョキ理論 : ふ

Xバー理論のこと。あらゆる文章を表現できる理論のため、明日から会話のたびに「はい、お前の発言、Xバー」「ナイス、Xバー」という返しができるようになる。いじめに使ってはいけない。
