陰謀論免疫
==========================================================
.. note:: 
    * 陰謀論や都市伝説などに傾倒した経験のある人が持つ免疫。
    * この免疫を持つものは、それっぽい学説を安易に信じない。
    * 陰謀論は麻疹（はしか）と同じ。一度はかかっておいた方がいい。
