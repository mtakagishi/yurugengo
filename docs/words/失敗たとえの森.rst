失敗たとえの森
==========================================
.. glossary::
    失敗たとえの森 : し

学者が一生懸命たとえたのに失敗した例えを味わうコーナー名

.. tip:: 
  半導体を巨大プリンに例えた失敗からの着想。命名は `名問の森 <https://amzn.to/3O7m0Wv>`_ からインスパイアされた
