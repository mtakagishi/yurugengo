ウェッジ―
==========================================
.. glossary::
    ウェッジ― : う

* パンツが尻の間に挟まって食い込んでいる状態
* 意図的に尻の間に食い込んだ状態にする行為を刺す英語の俗語

.. tip:: 

  被害者の頭や肩に届く程度にパンツを強く引き上げる行為をアトミックウェッジ―(atomic wedgie)という
