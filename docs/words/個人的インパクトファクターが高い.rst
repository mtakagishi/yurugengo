個人的インパクトファクター
=======================================
.. glossary::
  個人的インパクトファクターが高い : こ

* 論文がどれだけ引用されたかを示すインパクトファクターという概念を個人が読んできた本に適用する試み。
* 引用したり引き合いに出すことが多い本。
* 単に好きな本と違う。
* 影響を受けた本とも違う可能性がある。
* 本質本と被る部分がある。
