量子スプレマシー
==========================================
.. glossary::
    量子スプレマシー : り

現在の量子コンピュータは「ある種の問題においては速そうであると思われている」。この歯切れの悪い状況を打開し、ある種の問題において優位性があること（スプレマシー）の証明を目指す理論研究のことを量子スプレマシーという。
