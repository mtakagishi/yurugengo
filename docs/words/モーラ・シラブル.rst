モーラ・シラブル
==========================================
.. glossary::
    モーラ : も
    シラブル : し

モーラ、シラブル、共に、連続する言語音を区切る分節単位の一種。日本語はモーラの言語、英語などシラブルの言語

:シラブル: 音節、音声の聞こえの一種のまとまり
:モーラ: 拍、一定の時間的長さをもった音

.. tip:: 
  おじさん・おじいさん格付けチェックをやれば、本当にモーラの長さが分かるのか確認できる？
