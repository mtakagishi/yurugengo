衒学者
==========================================================
.. glossary::
  衒学者 : け

* 衒学とは知識をひけらかすこと。学を衒（てら）うこと。
* 衒学者とはpedant（ペダント）を語源とする。
* 秘められた裏のミーニングが衒学チャンネルで語られている。 ⇒ 関連動画

.. tip:: 
  パーソナリティの堀元氏が5年ぶり2022年4月20日に作った会社は『`株式会社pedantic <https://note.com/kenhori2/n/n32a21f7544c9>`_』
