ネクステージ
==========================================
.. glossary::
  ネクステージ : ね

* 受験英語向けに出版されている分厚い参考書であり、堀元氏が受験英語で用いた参考書である。

.. tip:: 
  センター試験英語をガチ攻略したという堀元氏の点数は198点。水野氏「俺より点数高いのやめて」(#103【ターゲット1900①】にて)
