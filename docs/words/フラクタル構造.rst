フラクタル
==========================================
.. glossary::
    フラクタル : ふ

* 数学者ブノワ・マンデルブロが導入した幾何学の概念。自己相似性。図形の全体をいくつかの部分に分解していった時に全体と同じ形が再現されていく構造
* 言語のフラクタル性として雪の結晶に例えられることがある。たまにブロッコリーに例える人もいる。
