チェーンソー
==========================================
.. glossary::
    チェーンソー : ち

ゆる音楽学ラジオの浦下さんが行うのはチェロ演奏会なのかチェーンソー会なのか。

.. tip:: 
  チェーンソー会発言に対する浦下さん本人からのコメント『イベントのご案内とチケット買ってくださった方ありがとうございます！カフェの近くにホームセンターってありましたっけ、まずチェーンソーの使い方から... 』
