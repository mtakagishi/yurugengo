メガフェプス（MEGAFEPS）
==========================================
.. glossary::
    メガフェプス : め
    MEGAFEPS : A-Z

* 以下の頭文字

  * Enjoy
  * Give up
  * Avoid
  * Finish
  * Escape
  * put off
  * Stop
