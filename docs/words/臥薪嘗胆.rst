臥薪嘗胆（がしんしょうたん）
==========================================
.. glossary::
    臥薪嘗胆（がしんしょうたん） : か

苦労・苦心すること。薪で寝る痛みで屈辱を思い出し、苦い肝を嘗めることで屈辱を忘れないようにして、目的達成のために耐え忍ぶこと。
