プレゼントの価値
==========================================
.. glossary::
    プレゼントの価値 : ふ

現金やアマゾンギフト券を贈る方が合理的。でもプレゼントを贈る理由は「その人のことを考える」ということに意味がある

.. tip:: 
  ということはプレゼントクリシェだと思う（堀元さん）
