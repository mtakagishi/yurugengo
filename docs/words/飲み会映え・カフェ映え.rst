飲み会映え・カフェ映え
==========================================
.. glossary::
    飲み会映え : の
    カフェ映え : か

日本語の「おもしろい」には、英語における"funnny"と"interesting"の2つの意味が包含されている。そのおもしろさを分ける言葉としてミーム提案された言葉。

飲み会映え::
  飲み会の場で楽しい陽気でお調子者タイプのおもしろさ

カフェ映え::
  カフェで1:1でじっくり話して楽しいタイプのおもしろさ
