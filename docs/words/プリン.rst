プリン
==========================================
.. glossary::
    プリン : ふ

トランジスタの説明をする際にデカいプリンで説明。つまりプリンと言えば半導体です。

.. tip:: 
  プリンの例えを思いついたとき「天才」を感じたが最終的に説明は散らかった。
