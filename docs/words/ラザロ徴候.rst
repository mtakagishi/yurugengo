ラザロ徴候
==========================================
.. glossary::
    ラザロ徴候 : ら

脳死の人が手足を動かす様子に対してつけられた言葉

.. tip:: 
  ラジオ内では生き返ったラザロに由来する言葉があるという荒い引用で登場
