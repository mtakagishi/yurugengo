IT原始人
==========================================
.. glossary::
    IT原始人 : A-Z

ITに疎い人々。パソコンがほとんどわからず、メモの取り方、メールの仕方などにも苦戦する人々が呼ばれがちな蔑称。「はじめての〇〇」などのソフトの解説本を頼りにしがち。

.. tip:: 
  ワードにメモを取る習慣だった水野氏。スプレッドシートというのがどうやらあるらしいぞ？便利らしいぞ？ということでスプレッドシートにまとめるようになった。堀元氏より相変わらずIT原始人ですね。
