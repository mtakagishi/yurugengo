俺たちのベスト方言グランプリ
==========================================
.. glossary::
    俺たちのベスト方言グランプリ : お

標準語に対するタカ派高村光太郎としてのコントリビューション企画。

.. tip:: 
  * つまり、標準語に加えたら便利な方言を視聴者から募集して紹介するという企画です。
  * 第一回では殿堂とグランプリ、第二回では水野賞、堀元賞、大賞が発表されました。
