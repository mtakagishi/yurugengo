ウミガメのスープ
==========================================
.. glossary::
    ウミガメのスープ : う

シチュエーションパズル、水平思考パズル、Yes/Noパズル。出題の真相を「はい」か「いいえ」で答えられる質問に対する回答を手掛かりにして推理する遊び。

.. admonition:: 有名な例題 [#u1]_
  
  ある男が「ウミガメのスープ」を一口飲み「これは本当にウミガメのスープですか？」「間違いございません。」男は帰宅した後、自殺をしました。何故でしょう？

.. admonition:: ゆる言語学ラジオの場合

  「赤ちゃんは言語を修得する。なぜでしょう」
