禿頭（とくとう）
=====================================
.. glossary::
    禿頭（とくとう） : と

禿げた人。禿げと直接言うと差しさわりがあるので憶えておくと便利な言葉。瞬時に言われると脳内で変換できない。

.. tip:: 
  方言グランプリにて「はげ」という方言が紹介された。（禿頭のことではない）
