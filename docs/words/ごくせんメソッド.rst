ごくせんメソッド
==========================================
.. glossary::
    ごくせんメソッド : こ

ヘラヘラしてたらブン殴られる、そんな衝撃のギャップで相手を黙らせる方法

.. tip:: 
  * かわいい若い女教師だと思ってなめてたら、極道の娘ヤンクミについていきますと改心する生徒という構図からの例え
  * 例文のマジシャン定延利之先生のごくせんメソッドに快楽を覚えた堀元氏は自分のフェチを再発見した。
