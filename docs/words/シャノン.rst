シャノン
==========================================
.. glossary::
    シャノン : し

* 情報をエントロピーで数値化する理論を考えた人。
* 来たる未来、友達の情報量を測るエントロピースカウターが登場するかもしれない。「エントロピーたったの５か、ゴミめ・・・」

.. tip:: text
  * シャノンの情報量は教養悪口本に登場する悪口の一つ。
  * ゆるコンピュータ科学でもテーマにされている。
