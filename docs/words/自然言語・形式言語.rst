自然言語・形式言語
==========================================
.. glossary::
    自然言語 : し
    形式言語 : け

:自然言語:
  日本語、英語のように自然に使っている言語。曖昧さが残されたり、話者集団によって用法がうつろいゆく
:形式言語:
  プログラミング言語のように、用法の変化に関して厳格に一義的に決まる人工的な言語。
