背中ブレイク
==========================================
.. glossary::
    背中ブレイク : せ

背中に「魔女の一撃」を受けて、腰から下が不随になるような状況。これをゆうたら「背中ブレイク！」

.. tip:: 
  魔女の一撃は、ドイツ語でHexenschuss：ヘクセンシュース
