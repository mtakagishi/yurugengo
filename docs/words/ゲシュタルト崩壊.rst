ゲシュタルト崩壊
==========================================
.. glossary::
    ゲシュタルト崩壊 : け

ゲシュタルトとはドイツ語で「形態」のこと。ゲシュタルト崩壊は形態認知についてヒトに発生しがちなバグ。漢字や仮名に対して長時間凝視や大量に書き取りで体感することがある。
