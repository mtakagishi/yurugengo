富士谷成章
==========================================
.. glossary::
    富士谷成章 : ふ
    ふじたに なりあきら : ふ

* 江戸時代に品詞分類を最初にした天才的言語学者
* 偶然にも、係り結び「こそ」の解釈について富士谷成章と同じ解釈を行った堀元さんをみて、蔦先生から「堀元さんの前世は富士谷成章に確定」。

.. tip:: 調子がいい時は「あきら100%」、調子が悪ければ「あきら0%」
