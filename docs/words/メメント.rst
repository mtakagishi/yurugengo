メメント
==========================================
.. glossary::
    メメント : め

ゆる言語学ラジオは、実際の撮影と動画公開時系列がバラバラであるため、視聴者目線でしばしば意味不明な会話の流れが発生する。こんな様子から忘年会ライブで視聴者から「時系列どうなってるんだよ、メメントかよ」とコメントされるに至る。

.. tip:: 
  メメントとはクリストファーノーランによる革新的なアメリカ映画。分割されたシーンが本来の出来事と交互に映し出され時系列メチャクチャな形式を取っている。
