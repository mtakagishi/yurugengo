おじさん・おじいさん格付けチェック
==========================================
.. glossary::
    おじさん・おじいさん格付けチェック : お

おじさん（アクセントは平板）と、おじいさん（アクセントは「じ」）、あえてアクセントを入れ替えてみると不思議と聞き分けが難しくなるので面白い。

.. tip:: 
  * 日本語は、モーラ（音の拍の長さ）よりもアクセントで聞き分けてる。
  * 「おまえ結局アクセント頼りじゃん」って煽れる

.. tip:: 
  ぜんいん、たいくかん、など漢字に当てはめて読むとモーラが違う言葉も興味深い
