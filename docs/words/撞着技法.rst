撞着技法
==========================================
.. glossary::
    撞着技法 : と

矛盾した言葉を二つ並べるシェイクスピアがよく使ったとされる。「冷たい熱血」のような矛盾する表現をあえて取り入れる技法

.. tip:: 
  ゆとたわが頻出句「それじゃ、こんばんは、おやすみなさい」の話題で補足された。
