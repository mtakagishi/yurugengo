白髪三千丈
==========================================
.. glossary::
    白髪三千丈 : は

1. 唐の詩人李白による心配や悲しみが深いことのたとえ。
2. 中国人の極端な誇張表現の例。誇張表現に対する暗喩として用いられる。

.. tip:: 
  こいつ盛ってるなと思ったら「はい、白髪三千丈！」
