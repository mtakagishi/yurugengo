途方もなく間違っていた
==============================
.. glossary::
  途方もなく間違っていた : と

* :doc:`ゆる言語学ラジオ` で多用される形容詞。
* :doc:`OED` へのリスペクトを想起させる。
* OED編纂の見積りは「途方もなく間違っていた」
* 略して「とほまち」
