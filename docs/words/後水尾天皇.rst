後水尾天皇
==========================================
.. glossary::
    後水尾天皇 : こ

ミズノの音を含むことに加えゴミが冒頭に加わった響きで高校時代はイジられていた。

.. tip:: 
  
  水野さんが高校時代に英語を教えていた一軍女子のいるLineグループの名前。
