トキトキ・トキントキン
===========================
.. glossary::
  トキトキ・トキントキン : と

* トキントキンともいう
* 削りたての鉛筆のような尖ったイメージを表現する形容詞。
* 愛知（名古屋や三河）の方で使われる方言。
* #27 でそっとスルーされ、#28 でツッコミが入り、#61で語源の説が紹介された。
