天丼
==========================================
.. glossary::
    天丼 : て

同様のボケを繰返すことで笑いを取る手法

.. tip:: 
  第一回赤ちゃんミステイクアワードの大賞の受賞は「天丼」をわかっているコロ助さん（違）の娘さんのエピソードでした。次回トロフィーを渡すのはピースケさん（違）に会場にきていただきトロフィーを渡すことになるのか？楽しみを残して第一回は終了した。
