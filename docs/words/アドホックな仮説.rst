アドホックな仮説
==========================================
.. glossary::
    アドホックな仮説 : あ

後付けの追加説明。その場しのぎの仮説

.. tip:: 
  * 地動説、フロギストン説などがクリシェ
  * 詳しくは「教養悪口本」参照 [#本1]_
  * 「た」第4回で登場する「た」の解釈に対し「それはアドホックな仮説なのでは」
