ガバガバの夏井いつき先生
==========================================
.. glossary::
    ガバガバの夏井いつき先生 : か

俳句について、俳句素人がガバガバな添削をする様子について、俳句添削で有名な夏井いつき先生のイメージを踏まえて表現した発言。決して夏井いつき先生のことをガバガバだと言っているわけではない。
