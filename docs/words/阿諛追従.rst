阿諛追従（あゆついしょう）
==========================================
.. glossary::
    阿諛追従（あゆついしょう） : あ

堀元氏「あ・・・鮎？？さかな？？？」　水野氏「教養がないやつは、これだから・・・正しくは曲学阿世かな？」
