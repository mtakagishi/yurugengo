ロバスト・robust
==========================================
.. glossary::
    ロバスト : ろ
    robust : A-Z

* 堅牢性を示す英語。コンピュータ科学の分野で頻出し、堅牢であればエラーやおかしな入力に対応できているということ。脆弱性の対義的な言葉にあたる。
* 言語学の分野の方にとって馴染みがない単語として登場する。
