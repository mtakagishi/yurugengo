タラン・チューニング
==========================================
.. glossary::
    タラン・チューニング : た

台本として用意した例え話の検討が不足気味で例えがイマイチな状況

.. tip:: 
  アラン・チューリングシリーズにて、チューリングの停止性問題の説明で例え話の失敗に一言「タラン・チューニング」ということですね（`動画 <https://www.youtube.com/watch?v=_slVM-J7t-0&t=2049s>`_）
