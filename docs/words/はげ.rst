はげ
==========================================
.. glossary::
    はげ : は

奄美大島で使われる感嘆詞。禿頭のことではない。

:(例): 「ベスト方言グランプリに選ばれたよ」「はげ！すごいがね！」
:(例): 「赤点取った」「はげ！勉強せんかったわけ？」
:(例): 「入院したんだよね」「はげ～。大丈夫ね？」
