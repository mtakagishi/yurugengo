ちーちーかーかー
=====================================
.. glossary::
    ちーちーかーかー : ち
    ぺーちんぺーちん : へ

沖縄の方言、「ちーちーかーかーする」と動詞で使う。一部の地域では「ペーちんぺーちん」とも言う。

.. tip:: 
  「口の中の水分が食べ物などの食べ物などにより亡くなった状態」として紹介されたが動画後半では早くもドリフトの気配か。
