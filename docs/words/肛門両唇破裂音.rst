肛門両唇破裂音
==========================================
.. glossary::
    肛門両唇破裂音 : こ

子音の'p'（プ）の音で、特に肛門で発生する空気の膨張によりでる破裂音。

.. tip:: 
  言語学者は、言葉としての子音の'p'（プ）を両唇破裂音と呼んでおり、それにちなんでおならの音をふざけてこのように呼んでいるらしい
