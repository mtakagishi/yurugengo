肛門両唇破裂音
==========================================
.. glossary::
    肛門両唇破裂音 : こ

子音の'p'（プ）の音で、肛門で発生する空気の膨張によりでる破裂音。

.. tip:: 
  言語学者界隈では両唇破裂音の'p'（プ）にちなんでこう呼んでいるらしい
