た
==========================================
.. glossary::
    「た」 : た

.. image:: /_static/た.png

「た」がわからない。過去の助動詞「た」の存在意義はどこにあるのか？そんな素朴な疑問をぶつける日本語勉強中の架空の外国人（水野氏）とそれに対応するミスターホリモト（堀元氏）をみてリスナー全員が思ったことだろう。「オレぼーっと母語話者やってたんだ。」

.. tip:: 
  ちょっと「た」を調べようと思った水野氏、結果的に文献を13冊と論文まで読んだ結果「分かんなかった」話をラジオで展開する
