大泉洋
==========================================
.. glossary::
    大泉洋 : お

* （ゆるコンピュータ科学ラジオにおいて）インターネット通信で発生する「パケット」のこと。

.. tip:: 
  インターネット通信を「水曜どうでしょう」で例えたことでパケット一つ一つが「大泉洋」となり、通信時には大人数の「大泉洋」が一斉に旅に出て行ったり戻ってきたり消えてしまったり！？。

.. tip:: 
 大泉洋は分かりづらいのでパケットで例えて説明すると分かりやすくなることがある。
