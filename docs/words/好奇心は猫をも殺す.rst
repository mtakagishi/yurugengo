好奇心は猫をも殺す
==========================================
.. glossary::
    好奇心は猫をも殺す : こ

イギリスのことわざ。容易に死なないとされる猫も好奇心がもとで命を落とすことがあるということから『過剰な好奇心は身を滅ぼす』ということ。
