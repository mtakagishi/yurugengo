ミスターソーリー・Mr.Sorry
==========================================
.. glossary::
    ミスターソーリー : み
    Mr.Sorry : A-Z

水野氏がロンドン留学でつけられたあだ名。飲み過ぎて、メチャクチャ気持ち悪いダンスを踊りながら「I'm Sorry」と謝りまくっていたことに由来。
