プロクルステスの寝台
==========================================
.. glossary::
    プロクルステスの寝台 : ふ

無理矢理、基準の部分をずらして一致させること。

.. tip:: 
  ギリシャ神話にて強盗プロクルーステースが行っていた捉えた人をベッドの大きさに合わせて、足を伸ばす・切るなどした話から転じた表現
