暗渠は合流する
==========================================
.. glossary::
    暗渠は合流する : あ

暗渠とは地下を通る見えない水路。タモリさんが好きなヤツ。暗渠はどこかの川に合流する様子から、変化の着地点が合流する比喩で使われる。

.. tip:: 
  インド・ヨーロッパ語族で起きたドリフトについて、サピアは同じ源流が暗渠を通るが最終的に合流するという例えで説明した。
