タッパーが叫ぶ
==========================================
.. glossary::
    タッパーが叫ぶ : た

Siriみたいなものが内臓されたタッパーに食べ合わせの悪い食材で作り置きするとタッパーが叫ぶ。一説によるとノームチョムスキーの非常に重要な指摘なのだが、その出典は曖昧であり真偽のほどは定かではない。

.. tip:: 
  句構造規則を作り置きタッパーに例えた際の悪ふざけである。
