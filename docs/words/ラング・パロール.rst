ラング・パロール
===========
.. glossary::
  ラング・パロール : ら

ソシュール回で登場、言語はラングとパロールに分かれる

ラング
  * ラングは意味を理解するための規則、語彙や文法規則、言語の集まり
  * 相手に意味が通じないときは、相手のラングにないことにするのが便利。

パロール
  * パロールは個人の具体的な発話行為。
  * ラングの話をしていること自体がパロールである。
