ファナティック
==========================================
.. glossary::
    ファナティック : ふ

狂信的であること、強い思い込み、おもしろ深読み。

.. tip:: 
  狂信という意味のファナティックとフォネティックコードのフォネティックが混同するときは、語源が「ファン」であることで解決
