オデュッセイアメソッド
==========================================
.. glossary::
    オデュッセイアメソッド : お

手土産としてお酒を持っていくことで敵意がないことを示す手段

.. tip:: 
  動画でオデュッセウスの話を紹介した堀元氏の家には大量のお酒が蓄積されることになった。
