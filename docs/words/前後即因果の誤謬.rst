前後即因果の誤謬
==========================================
.. glossary::
    前後即因果の誤謬 : せ

* Aが起きた後にBが起きた時、因果関係を重視せずに原因をAだと判断する誤謬。
* 必要条件をみたすと十分条件も満たすと考えてしまう誤解。
