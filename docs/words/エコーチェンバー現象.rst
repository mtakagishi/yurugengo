エコーチェンバー現象
==========================================
.. glossary::
    エコーチェンバー現象 : え

* 風呂場のような閉じた空間で音が反響する物理現象から例え。
* 自分の意見や思想、自分が見つけた法則に対して肯定的な答えにしか出会わないと過学習してしまう。

.. tip:: 
  小林・益川理論は、世界がクォーク3つと言われる頃に、名古屋にはクォークが4種類あるという狭い世界で生まれた。エコーチェンバーにより、むしろ独自の考えが育っていくケースもある
