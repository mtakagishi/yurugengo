エコーチェンバー現象
==========================================
.. glossary::
    エコーチェンバー現象 : え

* 風呂場のような閉じた空間で音が反響する物理現象から例え。
* 自分の意見や思想、自分が見つけた法則に対して肯定的な答えにしか出会わないと過学習してしまう。
