バイオミメティクス・バイオミミクリー
==========================================
.. glossary::
    バイオミメティクス : は
    バイオミミクリー : は

生物模倣。自然の模型や構造、要素を模倣し技術開発に活かすこと。生物を意味する「Bio」、擬態・模倣「Mimetics」「Mimicry」の合成語。
