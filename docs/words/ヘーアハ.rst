ヒーヒ・ヘーアハ
==========================================
.. glossary::
    ヘーアハ : へ
    ヒーヒ : ひ

「はい」を古英語の発音すると「ヒーヒ」や「ヘーアハ」となる。相槌としての「はい」を古英語風にすると「ヘーアハ」。
