換骨奪胎(かんこつだったい）
==========================================
.. glossary::
    換骨奪胎(かんこつだったい） : か

「ゆる言語学ラジオ」をベースにした「ゆる○○ラジオ」のように、元の作品の発想や様式をベースに新たな創作を行うこと。

.. tip:: 
  元の意味は、身も心も入替て生まれ変わることを示す中国の言葉。
