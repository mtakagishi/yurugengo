でまかせ力
==========================================
.. glossary::
    でまかせ力 : て

アドリブでそれっぽく正確そうに言う能力。知恵の瞬発力。正確とは限らない

.. tip:: 
  でまかせ力の秘訣は、よくわからない講演会でパネラーをやり続けること
