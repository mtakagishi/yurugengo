母語話者なのに？
==========================================================
.. glossary::
  母語話者なのに？ : ほ

* 	母語話者であるのに母語についての説明ができない状況への煽り言葉

.. tip:: 
  * 2021年度 :doc:`ゆる言語学ラジオ流行語大賞` ベスト5入り
  * 学校の国語の点数が悪いと言われがち
