ノーパン
==========================================
.. glossary::
    ノーパン : の

意図しないダジャレ。

.. admonition:: 使用例

  :今のはノーパンだから: 今のはダジャレのつもりじゃないから
  :ノーパン出ちゃいました: 無意識にダジャレが出ちゃいました。
  :今日ノーパンです: 今日は下着を履いていません。

.. tip:: 
  | 「no pun intended」という英語の決まり文句の略
  | punは「駄洒落や語呂合わせ」
  | intendは「意図する」
