アンビピスト・ビーピスト
================================
.. glossary::
  アンビピスト : あ
  ビーピスト : ひ

:Unbepissed（アンビピスト）: まだ尿をかけられていない状態
:Bepissed（ビーピスト）: アンビピストの逆の状態。

.. tip:: 
  我思う、ゆえに、Unbepissed（アンビピスト）あり　（助数シーズハイ）
