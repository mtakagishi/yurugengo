小林・益川理論
==========================================
.. glossary::
    小林・益川理論 : こ

ニュートリノに関するノーベル賞で有名な小林・益川理論を腐女子の二次妄想と紐づけた大好評深読み回。視聴者からも多くの共感を読ぶ回となり、しばしば「小林・益川理論」回として言及されるようになった。
