鼻から悪魔
==========================================
.. glossary::
    鼻から悪魔 : は

鼻から悪魔が飛び出すかもしれない。すごく予想できない事が起こることの失敗例え。ログラミングにおける未定義動作を指す。

なにその「上からマリコ」みたいなやつというツッコミを受け、鼻から悪魔を歌いだすサディスティックな堀元さんは `こちら <https://youtu.be/h-R6wQXB6oI?t=56s>`_
