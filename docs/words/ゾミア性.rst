ゾミア性
==========================================
.. glossary::
    ゾミア性 : そ

ゾミアに住む民が国家の支配から逃れるために取る戦略
  * 国家から距離を置く
  * 定住を避けるために農業を嫌い、遊牧・採集・移動耕作で生活する。
  * 権力や富・知識の集中を避け、シェアリングを尊重する

ゾミア性
  * 何かの支配から一定の距離を取る傾向
  * そのための戦略を取ろうとすること、自己野蛮化
  * ゾミアの民っぽさ。ゾミア100%
