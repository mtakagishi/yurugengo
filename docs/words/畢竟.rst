畢竟（ひっきょう）
==========================================
.. glossary::
    畢竟（ひっきょう） : ひ

これをつけておけば説得力マシマシの副詞。

.. tip:: 
  仏教用語。「要するに」「結局」「つまり」といった副詞の強いバージョンで、語意としては「さまざまな経過を経ても、最終的には一つの結果に落ち着くところ」
