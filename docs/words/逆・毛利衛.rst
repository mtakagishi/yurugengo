逆・毛利衛／順・毛利衛
==========================================
.. glossary::
    逆・毛利衛 : き
    順：毛利衛 : し

エスペラントを考案したザメンコフ「人間なんていないんだ、いるのは、ロシヤ人やポーランド人やドイツ人やユダヤ人だけだ」に対し、宇宙飛行士・毛利衛さんの「宇宙から見た地球に国境がないことを発言された」の発想と類似しているとして登場した。順なのか逆なのかよくわからないがラジオでは「逆・毛利衛」がよく用いられる。
