ゆる言語学ラジオメモ
=============================================
.. rubric:: ゆる言語学ラジオとは
* 堀元氏と水野氏がゆるく楽しく言語の面白さを語るラジオです。
* アカデミズムの二次創作をコンセプトに、専門家ではない人々が言語学を面白がるコンテンツです。
* 分野を超えあらゆる学問を楽しく学べる世界を目指し2022年1月1日に姉妹チャンネル『ゆるコンピュータ科学ラジオ』を開始しました。
